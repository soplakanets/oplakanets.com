## oplakanets.com

Minimal static personal site for **Serhiy Oplakanets**.

### Overview

- **Type**: Single `index.html`, no CSS, no JS.
- **Content**:
  - Very short **About** section.
  - **Links** to selected work and profiles.
  - **Contact** section (email and/or other preferred channels).
- **Audience**: Humans and search engines; HTML will use clear semantic tags and basic metadata.

### Structure

- `index.html` – single HTML document containing:
  - `<head>` with title, meta description, and basic SEO/social tags.
  - `<main>` with three sections: About, Links, Contact.

There is no build step or dependency tree: open `index.html` in a browser or serve it from any static host.
